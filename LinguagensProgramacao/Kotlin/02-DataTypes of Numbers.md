# DataTypes of Numbers

```kotlin
val inteiro: Int = 1
// Todos os números de ponto flutuante são iniciados como Double
// Para iniciar um float é necessário digitar um f ou F no final
val float: Float = 1.3F
val double: Double = 1.4
val string: String = "Vitor"
val boolean: Boolean = false;
val char: Char = '1'
val long: Long = 121_212_121_212_121_212_112_121_121;
val short: Short = 12;
```
